# Comfyui_VPA_Editor

这是一个专为 ComfyUI 设计的图像标注与编辑工具插件，旨在提供更灵活、更独立的视觉提示词锚点）编辑体验。

## 🌟 核心特性

- **独立编辑器窗口**：拥有专属的 UI 界面，不再挤在原生的图像菜单中。
- **多图层管理系统**：支持多达 6 个自定义图层，每个图层均可独立显示/隐藏、删除和命名。
- **丰富的笔刷库**：内置多种基础形状，并支持从 `svg_brushes` 目录加载自定义 SVG 矢量笔刷。
- **实时同步预览**：节点内置双预览区域（原图预览 + 标注预览），编辑器的修改可实时应用到节点预览中。
- **高级图像保存**：集成高级保存节点，支持自定义路径、无元数据导出及灵活的命名规则。

## 🧩 包含节点

### 1. VPA图像加载 (VPA Editor Load Image)
该插件的核心入口节点。
- **输入**：接受图像输入，支持手动设置保存名称。
- **功能**：点击“用 VPA Editor 打开”即可开启独立编辑窗口。
- **输出**：输出原图、遮罩、文本名称、6个独立标注图层以及最终的合并图层。

### 2. VPA图像合并 (VPA Editor Merge Image)
用于将两个图层进行高精度合成。
- **逻辑**：将 `image1` 叠放在 `image2` 之上。
- **要求**：两张图像尺寸必须完全一致。

### 3. VPA图像保存 (VPA Image Save)
基于高级图像保存逻辑，专为 VPA 流程设计。
- **特性**：
    - 去除所有 ComfyUI 工作流元数据（保护隐私/减小体积）。
    - 支持自定义保存路径（绝对路径或相对于 output 的相对路径）。
    - **自定义命名模板 (`filename_pattern`)**：支持占位符动态命名。
        - `{name}`: 节点左侧输入的名称（默认为 untitled）。
        - `{time}` / `{date}`: 当前精确时间或日期。
        - `{random}`: 5位随机字符。
        - `{index}`: 批次序列号（多图保存时自动递增）。
    - **自动分类目录**：支持在模板中使用 `/` 自动创建子文件夹，例如 `{date}/{name}`。
    - 支持 PNG、JPG、WEBP 格式，且完全去除工作流元数据。

## 📂 存储说明

- **图片与数据**：所有标注结果和图层配置（JSON）均存储在 `ComfyUI/output/VPA_PNG` 目录下。
- **自定义笔刷**：您可以将自己的 `.svg` 笔刷文件放入插件目录下的 `svg_brushes` 文件夹中，重启或刷新后即可在编辑器中使用。

## 🛠 安装方法

1.  进入您的 ComfyUI 插件目录：`ComfyUI/custom_nodes/`。
2.  将 `Comfyui_VPA_Editor` 文件夹放置于此。
3.  重启 ComfyUI。

## 💡 使用技巧

- **自动命名**：当您将图片连接到“VPA图像加载”节点时，它会自动读取图片的文件名作为编辑名称。
- **图层混合**：编辑完成后点击“应用”，标注层会自动同步回 ComfyUI 流程中。
